Modificacion del contenido Web

  1. Cambiar el Texto de un Elemento: Selecciona un elemento de texto y cambia su contenido dinámicamente utilizando JavaScript.
  2. Agregar un Nuevo Elemento: Crea un nuevo párrafo (<p>) y agrégalo al final del cuerpo (<body>) de la página.
  3. Modificar Atributos de Imagen: Selecciona una imagen y cambia su fuente (src) dinámicamente al hacer clic en un botón.
  4. Eliminar un Elemento: Agrega un botón que, al hacer clic, elimine un elemento específico de la página.
  5. Cambiar Clases Dinámicamente: Asigna un event listener a un botón. Al hacer clic, cambia la clase de un elemento, alterando su estilo.
  6. Crear una Lista Dinámica: Crea una lista (<ul> o <ol>) vacía. Agrega elementos a la lista utilizando un formulario y JavaScript.
  7. Actualizar Contenido de Formulario: Crea un formulario simple y utiliza JavaScript para actualizar dinámicamente el contenido en tiempo real mientras el usuario escribe.
  8. Estilo Dinámico al Pasar el Ratón: Asigna event listeners a varios elementos. Cambia el estilo de cada elemento cuando el ratón está sobre él y restaura el estilo original cuando el ratón sale.
  9. Animación al Hacer Clic: Asigna un evento a un botón para que, al hacer clic, un elemento experimente una animación o cambio de estilo gradual.
  10. Actualizar Contenido con Datos Externos: Utiliza la función fetch para obtener datos de una API externa y actualiza dinámicamente el contenido de la página con esos datos.
